WAIL

Web Archiving Integration Layer
One-Click User Instigated Preservation


What Is It?

Web Archiving Integration Layer (WAIL) is a graphical user interface (GUI) atop multiple web archiving tools intended to be used as an easy way for anyone to preserve and replay web pages.

Tools included and accessible through the GUI are Heritrix 3.2.0 and Pywb 0.33.0.

Usage

For More Detailed Usage Information Visit The Wiki

MacOS

  1. Download and mount the DMG
  2. Drag the WAIL icon from the disk image to the /Applications directory
  3. Launch WAIL.app

Linux

  1. Download and unzip the file
  2. Drag the WAIL folder to a location of your choosing
  3. Launch WAIL

Windows

  1. Download and unzip the file
  2. Drag the WAIL folder to a location of your choosing
  3. Launch WAIL.exe
